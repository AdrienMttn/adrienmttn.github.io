CONTEXT

Dans le cadre de ma formation en BTS Services Informatiques aux Organisations (option SLAM), j'ai du réaliser une application WEB en groupe de 5 sur un thème de notre choix, dans le quelle j’étais chef de projet.

OBJECTIFS

L'objectif principal était de concevoir une application web permettant aux utilisateurs de découvrir le top 20 des musiques de la semaine. Le défi technique résidait dans l'agrégation de données dynamiques provenant de sources externes ( API YouTube ) tout en offrant une interface utilisateur moderne et fluide. Avec les fonctionnalités suivante :

  • Voir le top 20 de la semaine actuelle
  • Naviguer entre les semaines précédente
  • Écouter de la musique

COMPETENCES

POINTS DE RÉALISATION

Nous avons utilisé Vue.js pour le frontend, ce qui nous a permis de créer une interface utilisateur dynamique et réactive. Pour le backend, nous avons utilisé Node.js avec Express pour créer une API qui récupère les données de YouTube. Nous avons également utilisé SQL pour stocker les données des utilisateurs et leurs préférences musicales.

DIFFICULTÉS

L'une des principales difficultés à été de trouver un moyen de récupérer de manière automatique les musiques en tendance
